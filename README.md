# XhsEmoticonsKeyboard

> j.s 🇨🇳

也许是最良心的开源表情键盘解决方案。

# Features

* API > 9 
* 表情键盘支持无闪烁自跟随系统软键盘高度,及支持自定义高度
* 表情支持自定义格式,支持任意来源
* 组件支持完全自定义,样式支持任意更改
* 默认微信键盘样式
* 赠QQ键盘高仿,不谢

# Gradle Dependency

Users of your library will need add the jitpack.io repository:

```xml  
allprojects {
    repositories {
        jcenter()
        maven { url "https://jitpack.io" }
    }
}
```

and:

```xml
dependencies { 
    compile 'com.github.w446108264:XhsEmoticonsKeyboard:2.0.0'
}
```
